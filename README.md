## Holochain setup

uses version 0.49-alpha1.  
enter the dna directory and use nix shell to run the conductor :

- nix-shell https://holochain.love
- hc package
- hc run


## UI setup

in another shell enter the ui directory.  
npm install / yarn install  
npm start / yarn start
